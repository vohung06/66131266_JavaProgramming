## Giới thiệu:

## Nội dung chính:
1. Các khái niệm cơ bản của Java: biến, toán tử, các câu lệnh điều khiển, vòng lặp...
2. Làm quen với lập trình hướng đối tượng trên Java.
3. Tìm hiểu cách nhập/xuất dữ liệu trên Java.
4. Làm việc với stream và file.
